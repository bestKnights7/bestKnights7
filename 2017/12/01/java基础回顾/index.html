java基础回顾

1.什么是面向对象？你是如何理解面向对象的？面向对象的特征有哪些？

  • 面向对象就是一种软件分析开发的思想，通过封装，继承，抽象，和多态来构建一个对象模型，
    这些模型有着自己独立的属性和行为（方法)，向外开放一个调用的方法，java语言就是一个面向
    对象的语言，我们通过不同的对象来解决不同的问题,提高了代码的复用性，将程序员从执行者转换为指挥者。面向对象的开发过程就是不断的创建对象，使用对象，指挥对象和维护对象与对象之间的关系。
  • 面向对象的特征有封装，继承和多态.
    1. 封装
      封装就是将对象的属性隐藏起来，仅提供对外的访问方式，
      封装的好处是：安全性 复用性
      封装的原则：将不需要提供的都隐藏，将属性隐藏提供对外的访问方式。
      在java中提供对类的成员设置了一定的访问权限，实现类中成员属性的隐藏。 
      private：表示私有的，只能被该类来使用。  
defalut：默认的表示只能被同包下的类使用  
protect：表示只能被该类和该类的子类所使用保护同包下的子类，不同包的子类  
public： 表示所有的类都能使用
